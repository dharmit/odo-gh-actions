clientset Package clientset is used to inject clients inside commands To use this package: From a command definition, use the `Add` function to declare the clients needed by the command Then, from the `SetClientset` method of the `Runnable` interface, you can access the clients To add a new client to this package: - add a new constant for the client - if the client has sub-dependencies, define a new entry in the map of sub-dependencies - add the packages's client to the Clientset structure - complete the Fetch function to instantiate the package's client
