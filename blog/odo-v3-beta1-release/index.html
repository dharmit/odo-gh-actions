odo v3.0.0-beta1 Released

July 1, 2022 ¬∑ 4 min read
Armel Soro

3.0.0-beta1 of odo has been released!

Notable Changes‚Äã

Features‚Äã

Check this Playlist for an overview of the most notable changes in this release: https://www.youtube.com/watch?v=yTUk_rx3aP8&list=PLGMB2PY4SNOrBQabcLZ_M5rN8l5u0B_cw

Ability to show (and stream) logs of running component, with odo logs and odo logs ‚Äìfollow (#5622, #5715)‚Äã

Substituting variables into the Devfile from the CLI when running odo dev or odo deploy (#5489)‚Äã

Support for composite run and debug Devfile commands when running odo dev (#5054)‚Äã

Ability to list existing Service Bindings and interactively generate Service Binding from a directory with no Devfile (#5635, #5772)‚Äã

‚ÄúBind as files‚Äù UI update when running odo add binding (#5770)‚Äã

Ability to unbind ServiceBindings from the Devfile, with odo remove binding (#5693)‚Äã

Bug fixes‚Äã

  • Better support for command and args fields in Devfile container components: odo no longer overrides the container command with Supervisord (#5648)
  • Devfile Volume components shown when personalizing Devfile configuration via odo init (#5779)

odo.dev‚Äã

  • Main site switched to 3.0.0 documentation
  • Blog post about binding an external service with odo v3 (link)
  • Homebrew installation details (#5801)
